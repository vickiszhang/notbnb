
### Starting the backend server

1. create .env file in backend

HOST="..."
USER="..."
PASS="..."
PORT="..."
DBNAME="..."

2. cd backend
3. run "node index.js"

### Starting the frontend-react server
1. cd frontend-react
2. run "npm start"

### Starting the frontend server
1. cd frontend
2. run "npm run dev"
